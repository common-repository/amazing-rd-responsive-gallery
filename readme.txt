== Installation ==

<p>Installation of <strong>Amazing RD Responsive Gallery</strong> is very simple.</p>

<strong>Option 1</strong>

<li> Download the plugin amazing-rd-responsive-gallery.zip from download location.</li>
<li> Unpack the *.zip file and extract the Amazing RD Responsive Gallery folder.</li>
<li> Drop the Amazing RD Responsive Gallery plugin folder into your wp-content/plugins folder.</li>
<li> In word press administration panels, click on Plugins from the menu.</li>
<li> You should see your plug-in listed under Inactive plug-in tab.</li>
<li> To turn the word presses plug-in on, click activate.</li>

<strong>Option 2</strong>

<li> Download the plugin Amazing RD Responsive Gallery Plugin from download location.</li>
<li> Go to Add New menu under Plugins tab in your word-press admin.</li>
<li> Select upload link (top link menu).</li>
<li> Upload the available Amazing RD Responsive Gallery Plugin file and click install now</li>
<li> Finally click activate plug-in link to activate the plug-in.</li>

<strong>Settings Page</strong>	

<p>Under <strong>Settings -&gt; Amazing RD Responsive Gallery plugin</strong> you can configure the basic settings. Everything else can be done on the fly via Custom post named<strong> RD Gallery</strong> on your Dashboard.</p>

<p><strong>Add New, Amazing RD Gallery- </strong>Allows you to add hover feature to Custom Posts as well as add information such as:  Title, Description, Image, in the Amazing RD Responsive Gallery custom post.</p>


<p><strong>Item Position -</strong> You can select which Hover item you want to appear first.  You can also select all of your items and drag to move the order of each up and down in the Custom Post Panel. </p>
<p>

<strong>Category - </strong>Choose the Category as you want to use for your Gallery Items.</p>


<strong>How To Use</strong>

<p>When you go to your Wordpress Dashboard you will now see a new custom post named <strong>RD Gallery </strong>in your left toolbar under Pages in admin Dashboard.

<strong>pages.</strong> This will allow you to quickly create Amazing RD Responsive Gallery item on your site.</p>


<p>
To create an Amazing RD Responsive Gallery all you need is the Fill All setting. The image can come from any source and be any size, you can also use images uploaded right from the <strong>Post Media Uploader</strong>.  
</p>
